Voice Agent is a local voice-powered grocery and to-do list manager that runs entirely on your machine.

PRIVACY NOTICE:
This application processes all data locally on your machine. No data is sent to cloud services.
- Voice transcription happens locally using Vosk
- AI inference happens locally using Ollama
- All data is stored locally in JSON format
